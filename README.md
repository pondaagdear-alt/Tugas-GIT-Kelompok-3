# Tugas-GIT-Kelompok-3
Repository Tugas GIT Kelompok 3 Proyek Kolaborasi Pyhton "To Do List"

# Aplikasi To Do List
Aplikasi To Do List ini adalah aplikasi yang dibuat dengan tujuan mempermudah pengguna mengingat dan mengatur tugas yang akan dilakukan.
Aplikasi ini dibuat menggunakan bahasa pemrograman Python yang dapat di jalankan pada terminal/consol.


## Features

- Tambah Tugas 
- Tampilkan Tugas
- Edit Tugas
- Hapus Tugas
- Simpan Tugas
- Muat Data

## Cara Penginstalan

Langkah-langkah untuk mengunduh dan menyiapkan program:

1. Buka Python (Pastikan Phyton sudah terpasang)

* Cek dengan perintah:
```sh
python --version
```

2. Clone repository dari GitHub

* Buka Command Prompt/Terminal, Kemudian ketik:
```sh
git clone https://github.com/username/nama-repo.git
```
* Kemudian ganti "username/nama-repo" dengan Tugas-GIT-Kelompok_3
* Setelah itu, masuk ke folder program:

```sh
cd nama-repo
```
